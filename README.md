## 🧑‍💻 About Me

I'm **Prasad Hiwarkhede**, a developer from **Nagpur, Maharashtra 🇮🇳** with a passion for building software that solves real-world problems. My work spans artificial intelligence, desktop application development, cybersecurity tools, and full-stack web systems.

I've competed in international challenges like the **NASA Space App Challenge** as part of **Team Xebec**, and I'm deeply interested in the rapidly growing field of **Retrieval Augmented Generation (RAG)**. When I'm not writing code, I'm thinking about how technology can make everyday life more secure, efficient, and inclusive.

- 🔭 Currently working on **ycceRAG** — an AI-powered RAG system for contextual knowledge retrieval
- 🛡️ Building security-focused tools to protect users from online scams and threats
- 🖥️ Crafting native Windows desktop experiences with **.NET 8 + WPF + WebView2**
- 🌱 Exploring multi-modal RAG, LLM fine-tuning, and cross-platform desktop frameworks
- 🏆 NASA Space App Challenge participant · SDG9 Hackathon contributor
- 💡 Fun fact: I named my security monitor after **Heimdall** — the Norse guardian of the realms
